Installing Ubuntu 22.04
Minimum Requirements for Installing Ubuntu 22.04
  • RAM: 4GB RAM
  • 25 GB Free Disk Space
  • CPU: 2 Cores
  • Ubuntu ISO image
  • VirtualBox with Extension Pack

  2. To download, click the host operating system that you have in the VirtualBox Platform Packages section (lower left).
  3. After successfully downloading VirtualBox, in Windows, proceed to the installation by clicking the downloaded file and click next.
  4. Click "I accept the terms in the License Agreement.

  6. If prompted with a warning like below, just click Yes.
  7. If prompted with missing dependencies like below, click Yes.

How to install Ubuntu 22.04?
  1. Click the link here and you will be routed into this page where you will select and download Desktop Image ISO File. Once in the page, click 64-bit PC (AMD64) desktop image.
  2. After downloading the ISO file, you may now open your VirtualBox and click New.
  3. You may now be able to see this on your end:
  4. Input the name Ubuntu 22.04 for easy navigation and make sure you choose the folder where you have enough space.
  5. In ISO Image just click Other… and locate your ISO file.
  6. This is how it should look before proceeding into the Unattended Install.
  7. In this section you may now change the username and password.
  8. On this screen, you can specify how much RAM the virtual machine should get and also the number of processors. Keep in mind that as you allocate memory to your virtual machine, this will take it away from the host machine. That is why you see it orange and red up near the top. Do not worry about this as you can change it based on what you need anytime you want, as per now I want mine to take space like what is shown below.
  9. This part allows you to specify how much hard disk space you want for your VirtualMachine. It is recommended to click the Pre-allocate Full Size. Now, click Finish. Wait until the virtual machine has finished installing. Once it is done, your window should resemble the image below. Warning! If you are presented with a terminal, just navigate through your arrow keys to Ubuntu (Safe Graphics) and hit ENTER. This will open a window where you can see the installation of Ubuntu 22.04, this is an attended install. Just wait for the installation to finish which would take around 10 minutes (depending on your resource provision).
  10. If you encounter this error just restart your device and re-open your Virtual Machine.
  11. Now open your Ubuntu 22.04 by clicking Start.
  12. Your screen should look like this, now fill the necessary details for your log in.
  13. All you can do now is to wait for it to be fully downloaded.
  14. Congrats! 🎉 You’re now ready to explore Ubuntu! If the installation was tricky, feel free to ask—but hey, YouTube’s always there too! Now go have fun and create something awesome!

Installing ROS 2 Humble
How to install ROS 2 Humble?
The following installation instructions assume that you are using an x86_64 laptop or computer with Ubuntu 22.04 (Jammy Jellyfish). You can use virtual machines like VirtualBox or QEMU if you are using a different operating system. Installing with other Linux distributions has not been tested yet. Make sure you also have a stable installation of Python 3 and Git. Most of the code here follows the ROS 2 Humble Hawksbill documentation. The installation includes ROS 2, rqt, colcon, and all the other necessary packages or libraries you may need in completing this course. Tip: Type CTRL + ALT + T to open a terminal or CTRL + SHIFT + T to add a new terminal tab. You need a terminal to run the code below.
  1. Set locale. 
    locale  # check for UTF-8
    sudo apt update && sudo apt install locales
    sudo locale-gen en_US en_US.UTF-8
    sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8
    export LANG=en_US.UTF-8
    locale  # verify settings
  2. If the error user is not in sudoers file appeared, perform the code below. 
    su root
    Provide your password then the terminal will show a different prompt like root@user:/home/user. 
    nano /etc/sudoers
    Add your username to the sudoers file be. Change the username user to the username you used during installation. 
    user	ALL=(ALL) ALL
    Your terminal should look like the one below, replace user with the proper username. To save, type CTRL + O then ENTER, it should be saved by now. Exit with CTRL + X. Type exit to leave root.
  3. Setup sources 
    sudo apt install software-properties-common
    sudo add-apt-repository universe
    sudo apt update && sudo apt install curl -y
                                    
    sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg
                                    
    echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null
  4. Install ROS 2 Humble. 
    sudo apt update
    sudo apt upgrade
    sudo apt install ros-humble-desktop
    sudo apt install '~nros-humble-rqt*'
    sudo apt install python3-colcon-common-extensions
    sudo apt install python3-rosdep2
    sudo apt install ros-humble-rqt-graph
    sudo apt install ros-humble-turtlesim
    sudo apt install git
    rosdep update
    echo "source /opt/ros/humble/setup.bash" >> ~/.bashrc
  5. Verify installation. Open a terminal and run this code. 
    ros2 run demo_nodes_py talker
    Open a different terminal and run this code. 
    ros2 run demo_nodes_py listener
Congrats! 🎉 If the installation is successful, you should see the same messages being printed on both terminals. The talker node is publishing those messages to the listener node.
